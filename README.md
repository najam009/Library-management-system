# Library Management System

Desktop app for library operations with Java.

## Features
✅ Book Inventory Management
✅ Member Management
✅ Issue/Return Books
✅ Fine Calculation

## Tech Stack
- Java 8+, MySQL, Swing GUI, JDBC

## Setup
1. Import `database/schema.sql`
2. Add MySQL Connector JAR to lib/
3. Compile: `javac -cp .:lib/* src/**/*.java`
4. Run: `java -cp .:lib/*:src Main`
